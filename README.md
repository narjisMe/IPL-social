## Explication du code
Ce projet est un projet Node.js en TypeScript.  
Il a une fonction de validation d’email dans `src/services/email.service.ts` et des tests unitaires dans `src/tests/email.service.test.ts`.  
A chaque Pull Request, le pipeline installe les dépendances et run les tests automatiquement.
